# Public Assets

This folder contains static assets that are served by Next.js.

## Icon Files

Place your PromisePay icon file here as:
- `promisepay-icon.svg` (recommended)
- or `promisepay-icon.png`

The icon should be 28x28 pixels or larger (it will be scaled down).

## Current Usage

The PromisePay logo in the sidebar will automatically use `/promisepay-icon.svg` if it exists, otherwise it will fall back to displaying "P" text.
